# 🎓 SGPA & CGPA Calculator

A modern and interactive web-based calculator to compute SGPA and CGPA easily with a clean UI and smooth animations.

---

## 🚀 Features

- 📅 Select academic year (1–4)
- 📊 Dynamic subject & credit input
- 📝 Grade selection system
- ⚡ Instant SGPA calculation
- 📈 CGPA calculation with previous data
- 🎨 Beautiful animated UI
- 📱 Responsive design

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript

---

## 💡 How to Use

1. Select your academic year  
2. Enter number of subjects  
3. Add credits  
4. Select grades  
5. View SGPA & CGPA
